# Lentil & Sweet Potato Dal

![Spinach, sweet potato and lentil dal - BBC Good Foods](https://images.immediate.co.uk/production/volatile/sites/30/2020/08/spinach-sweet-potato-and-lentil-dhal-be8fae5.jpg){width="50%" align="right" loading="lazy"}

## Ingredients - Serves 4

|                                            |
| ------------------------------------------ |
| **1 tbsp** sesame oil / ghee               |
| **1** red onion                            |
| **1** garlic clove                         |
| **1 thumb** ginger                         |
| **1** red chilli                           |
| **1.5 tsp** ground turmeric                |
| **1.5 tsp** ground cumin                   |
| **2** sweet potatoes                       |
| **250 g** red split lentils                |
| **400ml** vegetable stock                  |
| **200ml** coconut milk                     |
| **80g** spinach                            |
| **4** spring onions                        |


## Method
1. Cook onion in the sesame oil / ghee for _~10 minutes_, until softened
2. Add garlic, ginger, chilli, turmeric & cumin, cook for _~3 minutes_, adding splashes of water if it sticks to the pan.
3. Add sweet potato and stir to coat
4. Add lentils, stock and coconut milk
5. Bring to a boil, season to taste, then cover and simmer for _20 minutes_
  a. The lentils should be tender, and the potato should just hold its shape.
6. Gently stir in the spinach
7. Serve with spring onions on top.
